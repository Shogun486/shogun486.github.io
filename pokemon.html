POKÉMON IDENTIFIER

This Android and Web application allows a user to take a picture from their device of a Pokémon and upload it for identification.

This project required collaboration between myself and a few classmates who wanted to create a unique app for Pokémon fans. Please check out some demos of this app below!

  • View Demos

Mobile UI



My team found the cropping feature (as shown in the demo above) to be essential for the user because this permits anyone to take a picture and instantaneously feed the algorithm an isolated picture of the Pokémon. Similar to how you can use an app to identify a song you don't recognize, with our app you can now identify any Gen-1 Pokémon!


Website



A huge challenge in creating this app was making sure the evolution chains were displaying correctly. Since a single Pokémon could evolve into many other ones (and could be a combination of several types), making sure the right information was being displayed was extremely important.
